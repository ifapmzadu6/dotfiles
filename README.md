# dotfiles
- Available on MacOS/Linux(ubuntu).

## Install
1. Install packages
    -  Linux(ubuntu):
        - `sudo apt install git vim`
    -  Mac
        - do nothing
2. Install dotfiles
    - `git clone https://github.com/ifapmzadu6/dotfiles.git .dotfiles && .dotfiles/install.sh`

## Update

1. Update
    - `.dotfiles/install.sh`
